基于github pages 搭建hexo博客主页

环境搭建

所需环境：centos7 git nodejs hexo
我使用的是centos系统，所以先记录我在centos上的搭建过程

git nodejs使用yum即可

git 安装

nodejs安装

版本选择，我选的是V6.x

hexo安装

初始化博客

选择合适的路径创建hexo自定义名字的文件夹，我使用的是hexo打算在这里面搞事情

INFO Start processing
INFO Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.
去提示的页面浪吧

换主题 使用github pages

我使用的是next
先安装主题吧

这样就下载到themes文件夹里面了
然后启用next，需要修改站点【我是blog】下的配置文件_config.yml

然后可以本地看看

然后配置一下git
ssh-keygen -t rsa -C “your email address”

连续按3个回车（密码默认为空），得到 id_rsa 和 id_rsa.pub 文件，在/root/.ssh 下说明生成成功

打开 Github，登录自己的账号后
点击自己的头像->settings->SSH And GPG Keys->New SSH key

将本地 id_rsa.pub 中的内容粘贴到 Key 文本框中，随意输入一个 title(不要有中文)，点击 Add Key 即可

输入以下命令

Are you sure you want to continue connecting (yes/no)?
yes
